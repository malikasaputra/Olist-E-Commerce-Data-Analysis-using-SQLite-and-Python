# E-commerce Data Analysis (Olist)

## Overview
This project analyzes an e-commerce dataset from Olist to understand sales performance, customer behavior, and product trends.

## Tools Used
* Visual Studio Code
* Python
* SQLite
* Pandas
* Matplotlib
* Seaborn

## Key Insights
1. Trend Revenue dari waktu ke waktu
2. Top 3 product 
3. TOP 3 city 
4. Distribusi harga product
5. Korelasi antara hubungan delivery dengan review score
6. Distribusi payment menggunakan 

## Dataset
Olist E-commerce Dataset (Kaggle)
